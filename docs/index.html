Why?

Because GUIs are bloated. The old Bash script was good, but Rust is instant.

Manifesto

  • Native Rust Engine: No more external indexers. The entire app is a single, static binary.
  • Keyboard Driven: Fuzzy find (skim) everything.
  • Universal: Plays local files and URLs (via yt-dlp) seamlessly.
  • Forgiving: Smart shortcuts (`-l`, `-a`, `-g`) that auto-play single matches or open menus if ambiguous.

Usage

Run mpv-music to launch the menu, or use args:

  • mpv-music -a "Daft Punk" (Filter by artist, instant-play if 1 match)
  • mpv-music -l "Gym" (Play playlist directly)
  • mpv-music "https://youtu.be/..." (Stream URL)
