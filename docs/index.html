Music for your terminal.
Local library management.
Powered by mpv, ffmpeg, fzf, and bash.

curl -sL https://raw.githubusercontent.com/furqanhun/mpv-music/main/install.sh | bash

Why?

Because GUIs are bloated. And it's painful to run find to search for files.

Manifesto

  • Fast: Written in pure Bash. Starts instantly.
  • Keyboard Driven: Fuzzy find (fzf) everything.
  • Hybrid: Plays local files and URLs (those supported by yt-dlp) seamlessly.
  • No Databases: I mean JSONL is kind of a Database.

Usage

Run mpv-music to launch the menu, or use args:

  • mpv-music -a "Daft Punk" (Filter by artist)
  • mpv-music -p (Play all shuffle)
  • mpv-music https://youtu.be/... (Stream URL)
